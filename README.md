# CameraSystem
2DCameraSystemは、コンソールゲームやCUIアプリケーションの作成を支援します。
マップの移動やカメラの動きを表現するには、面倒な処理をたくさん書かなければならないです。
そこで、それを簡略化するクラスを作った。
このクラスは、私のゲームを開発するためのものなので、自分流の部分も多々あります。
今後のバージョンアップで使いやすくしていきたいと思います。
